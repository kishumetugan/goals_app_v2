# לוח המטרות שלי 🎯

אפליקציית מובייל לניהול מטרות, פתקים ולימוד תורה — עובדת ישירות בדפדפן, ללא שרת וללא התקנות.

## תכונות

- 🎯 **מטרות** — קטגוריות צבעוניות, הערות לכל מטרה, סימון הושלם
- 📓 **NOTEPAD** — פתקים בתיקיות, כוכבית, חיפוש, מיון, שיתוף
- 📖 **לימוד תורה** — מעקב ספרים, פרק/עמוד, צילום עמוד
- 💾 **גיבוי ושחזור** — קובץ JSON מלא לכל הנתונים
- 🎨 **6 ערכות צבע** — ניתן לשנות בהגדרות
- 🗑 **סל מיחזור** — לכל סוג תוכן
- 📤 **ייצוא TXT** — בעברית תקינה (UTF-8 BOM)

## מבנה הפרויקט

```
goals-project/
├── index.html   ← מבנה הדף
├── style.css    ← כל העיצוב
├── script.js    ← כל הלוגיקה
└── README.md    ← קובץ זה
```

## הרצה מקומית

פשוט פתח את `index.html` בדפדפן — זהו.

> ⚠️ אם פותחים כקובץ מקומי (`file://`) ייתכן שהדפדפן יחסום טעינת קבצים חיצוניים.  
> פתרון: השתמש ב-Live Server ב-VS Code, או העלה ל-GitHub Pages.

## GitHub Pages

1. העלה את כל הקבצים ל-repository
2. הגדרות ← Pages ← Branch: main ← Save
3. האפליקציה תהיה זמינה בכתובת: `https://USERNAME.github.io/REPO`

## נתונים

כל הנתונים נשמרים ב-`localStorage` של הדפדפן — לא נשלח שום דבר לשרת.  
לגיבוי: הגדרות (☰) ← "צור קובץ גיבוי".

## טכנולוגיות

- HTML5 / CSS3 / Vanilla JavaScript
- localStorage לשמירת נתונים
- Canvas API לכיווץ תמונות
- ללא תלויות חיצוניות
