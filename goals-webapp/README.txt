
לוח המטרות - גרסת ווב / PWA

מה יש כאן:
- index.html
- manifest.json
- service-worker.js
- אייקונים

איך להשתמש הכי פשוט:
1. מעלים את כל הקבצים כמו שהם לאחסון סטטי (למשל Netlify / GitHub Pages / שרת רגיל).
2. פותחים את הכתובת בטלפון אנדרואיד.
3. בדפדפן Chrome לוחצים על תפריט שלוש הנקודות > הוספה למסך הבית.
4. מתקבלת אפליקציית ווב שנפתחת כמעט כמו אפליקציה רגילה.

הערה:
כדי שההתקנה תעבוד בצורה תקינה, צריך לפתוח דרך https ולא דרך קובץ מקומי.
